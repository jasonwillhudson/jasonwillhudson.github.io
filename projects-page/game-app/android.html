Beauty Diary

MySQL / Java / Android Studio

Download APK

Introduction

The Beauty Diary is an app for users to write their beauty journeys such as losing weight, skincare and notes for the makeup. This app allows users to post their beauty journeys as moments, so that users can browse through the posts they posted to see the change and improvement from the past. Users can also search the posts with keywords to review the posts from the past and delete the ones that the user does not want to see again. In addition, the username and password help to make the journey private and safe.



Login / Register

User has to enter both username and password to view the diary they wrote, which helps to secure the privacy. If a user wants to change username and password, a secure code is required to make sure it's the correct person.





Browse / New Post

User can either press the "+" button or shake the phone to add a new post. In the post editing activity, user can type the paragraph in the text box. User is allowed to add photo to the post by taking a new photo from camera or choose existed ones from the gallery. When the post button clicked, the post will be posted to the moment page. On moment activity, user can swipe up and down to browse through the posts, or click on the image to see the full size of the photos.





Delete / Undo / Search

When user wants delete a post, simply swipe left and click on delete. After the delete button is clicked, the system will pop up a snack bar with undo button for few seconds. If the undo is clicked, the post will be recovered to its original position, otherwise the data of the post will be deleted from the SQL database permanantly.
